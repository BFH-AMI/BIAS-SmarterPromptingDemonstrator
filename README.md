# BIAS Smarter Prompting Demonstrator

This work is part of the Europe Horizon project BIAS funded by the European Commission, and has received funding from the Swiss State Secretariat for Education, Research and Innovation (SERI).
All work from the BIAS Project: https://github.com/BFH-AMI/BIAS 

## Overview
This repositories contains the BIAS Smarter Prompting Demonstrator, an AI Agent framework that modifies a user's prompt to any third-party LLM to make the answer less biased. The agents are based on scientific findings from the BIAS project, which are linked below. The repository contains the demonstrator as well as the code and data from the scientific publications.
